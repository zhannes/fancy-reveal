Fancy Reveal

Built on Foundation version 4.0.5.

  • Syntactic sugar for controlling Foundation Reveal via deferreds.
  • Methods for showActivity/hideActivity.

Usage

Pass an element, you'll get back a reference to a FancyReveal object, .e.g 

        var $el   = $('#myModal'),
            modal = App.lib.fancyReveal($el);

When you want to open it, call: 
        modal.open();

To try it, and read about more features, click below.

Chaining with deferreds

The fancyReveal instance is passed to each function in the chain.

    modal
      .showActivity()
      .then(resizeForDemo) 
      .then(function(overlay){
        overlay.hideActivity();
      });

    /* Note: 
       function resizeForDemo(){} 
       is defined elsewhere. 
       It returns a deferred */
